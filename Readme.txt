Association fictive sur Nôtre planete : Projet d'évaluation. 

Pour evnoyer mon projet sur Github comme indiquer dans les consigne de l'évaluation, voila ce que j'ai fait :

La branch Commit init - dépots des 3 pages principale l'index/ login / signup.html et du dossier sass.

 Dépôt de ReadMe.txt : Branch Main
 Dépôt de evenements.html et son dossiers sass : Branch Main 
 Dépot du fichier texte de base avec les idée : Branch Update

Schéma:

	Commit init :
		Dossier assets 
			( CSS / Images )

		Dossier SASS
			( Atoms, base, components, utils. )

		Les 3 fichiez principaux SASS
			Style.sass ( pour l'index.html )
			Signup.sass ( pour Signup.html )	
			Login.sass ( pour login.html )

		Les Pages .HTML
			index.html ( Page principale )
			login.html ( Formulaire de connexion )
			signup.html ( Formulaire d'inscription )

	Deuxieme Commit :
		ReadMe.txt

	Troisieme commit :
		Certaine correction + Ajout de Mixins ( Factorisation de code.) + ajout de fonction.
		Evenement.html ( Nouvelle page HTML )
		event.sass ( Son fichier sass/css )

	Quatrieme commit : 
		Quelques modification 
		ajout du fichier texte d'idée de départ.
	
	Les derniers commit :
		certaines correction mineur.
